  • ¶

    Many D3 plugins are intended to be invoked by selection.call, meaning that they produce a function that can be configured and then passed to selection.call. For example, d3-axis provides a number of constructors (axisLeft, axisTop, etc.) that all produce an axis, which is a function. It is also an object with methods for modifying the scale and ticks.
